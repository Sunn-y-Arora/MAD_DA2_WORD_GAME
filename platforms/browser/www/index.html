GUESS THE WORD GAME



  • The User has to choose a character and complete the word, but prefixing and suffixing the word withthe same character.
  • He is given three choices to find the word.
  • If he finds the word then he is given a score according to no. of attempts left
  • And is given another word to find, until all the words aren't guessed coorectly
  • Once the user is failed to guess the word within the 3 chances the Game is over, however the game can be restarted
